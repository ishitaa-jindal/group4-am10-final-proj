Introduction

75 years after the founding of the UN, women and girls still face inequality. What factors are influencing this inequality and which can be adopted? Using data from the World Bank, OECD, and the United Nations, we perform an analysis and try explaining some of these trends using statistical analysis and visualizations. We focus on Europes top & bottom 3 countries to avoid distortion by structural differences between different regions of the world and identify key events that led to some major changes in the inequality patterns for our chosen countries - which are:

  • Best 3 in terms of GII: Netherlands, Denmark, Switzerland
  • Worst 3 in terms of Gender Inequality Index: Hungary, Latvia, Poland

The main objectives are:

  • Visualise the pay gap between men and women around the world
  • Compare participation in the job market between men and women
  • Determine which countries are best and worst in terms of gender equality in Europe
  • Determine female presence in government in the 6 countries
  • Look at the different dimensions that are used to calculate the Gender Inequality Index value

The data we analyse can be found in the data folder of the repository https://github.com/maddy-g6/group4-am10-final-proj

Hypothesis Testing

We build confidence intervals for male and female income and see if the difference is significant at a 95% confidence interval using hypothesis testing where our null hypothesis is: There is no difference between male and female income.

# create groups for our top and bottom 3 countries in Europe
top_3 <- c("Netherlands","Switzerland","Denmark")
bottom3 <- c("Hungary","Poland","Latvia")
our_6 <- c("Netherlands","Switzerland","Denmark","Hungary","Poland","Latvia")

# filter data for our 6 countries
filtered <- all_income_hdi %>% 
  subset(all_income_hdi$country %in% our_6) %>% 
  rename(female= income_female) %>% 
  rename(male = income_male) %>% 
  pivot_longer(4:5, names_to = "Gender", values_to = "Income")
  
#filtered

# build confidence interval 
filtered %>%
  group_by(Gender) %>%
  filter(!is.na(Income)) %>% 
  summarise(mean_income = mean(Income, na.rm = TRUE),
            sd_income = sd(Income, na.rm=TRUE),
            count = n(),
            se_income = sd_income/sqrt(count),
            t_critical = qt(0.975, count-1), 
            margin_of_error = t_critical * se_income,
            lower = mean_income - t_critical * se_income,
            upper = mean_income + t_critical * se_income
            )
